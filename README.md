# Lab 4 Assignment: Data Quality Assessment & Preprocessing

Assignment Description
This assignment focuses on applying data preprocessing techniques on a chocolate sales dataset.

Tasks
Task 1: Data Quality Issues
Identified issues such as incorrect data types and formatting problems in the dataset.

Task 2: Missing Values
Applied mean imputation to handle missing values and preserve dataset size.

Task 3: Outliers
Detected and removed outliers using the IQR method.

Task 4: Normalization
Applied:
- Min-Max normalization
- Z-score standardization

 Task 5: PCA
Checked correlation between features. PCA was not applied due to weak correlation.

Conclusion
The dataset was successfully cleaned and prepared for machine learning tasks by handling missing values, removing outliers, and normalizing features.
